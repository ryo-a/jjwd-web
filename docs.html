JJWD で配信しているJSONデータ

2017/12/11時点では 全国約1300箇所に存在する AMeDAS 観測点による 最新の気象データ をそのまま JSON 化したもののみを配信しております。
  • 1時間降水量 ( http://jjwd.info/data/amedas-rain-1h-recent.json )
  • 3時間降水量 ( http://jjwd.info/data/amedas-rain-3h-recent.json )
  • 24時間降水量 ( http://jjwd.info/data/amedas-rain-24h-recent.json )
  • 48時間降水量 ( http://jjwd.info/data/amedas-rain-48h-recent.json )
  • 72時間降水量 ( http://jjwd.info/data/amedas-rain-72h-recent.json )
  • 降水量全要素 ( http://jjwd.info/data/amedas-rain-all-recent.json )
  • 最大風速 ( http://jjwd.info/data/amedas-max-wind-speed-recent.json )
  • 最大瞬間風速 ( http://jjwd.info/data/amedas-max-instantaneous-wind-speed-recent.json )
  • 最高気温 ( http://jjwd.info/data/amedas-max-temperature-recent.json )
  • 最低気温 ( http://jjwd.info/data/amedas-min-temperature-recent.json )
  • 積雪 ( http://jjwd.info/data/amedas-snow-depth-recent.json )
  • 24時間降雪量 ( http://jjwd.info/data/amedas-falling-snow-24h-recent.json )
  • 累積降雪量 ( http://jjwd.info/data/amedas-falling-snow-total-recent.json )
また、各 JSON ファイルの末尾に .gz を付加すると、 gzip 圧縮された JSONデータが取得できます。
